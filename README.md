# NLP-exercise2

The objective of this project is two create two character-based prediction models adapted from Andrej Karpathy’s makemore model. The models are trained to generate Pokémon names in two languages: English and French. 

# Code Dependencies: 
- Python 3.11.4 

Libraries: 
- pandas: 3.0.1
- sklearn: 1.8.0
- numpy: 2.4.2
- permetrics: 2.0.0
- gensim: 4.4.0
- scipy: 1.17.1

#Data sources: 
- NRC-VAD Lexicon: v1 updated 2018 https://saifmohammad.com/WebPages/nrc-vad.html
- Catalan Floret embeddings: https://zenodo.org/records/7330331
- Catalan FastText embeddings: https://zenodo.org/records/4522041
