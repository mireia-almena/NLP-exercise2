# NLP-exercise2

The objective of this project is two create two character-based prediction models adapted from Andrej Karpathy’s makemore model. The models are trained to generate Pokémon names in two languages: English and French. 

# Code Dependencies: 
- Python 3.11.4 

Libraries: 
- torch: 2.9.1
- matplotlib: 30.10.7
- pandas: 2.3.3
